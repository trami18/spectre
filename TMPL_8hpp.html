Function Documentation

◆ as_pack()

template<typename List , typename F >
constexpr decltype(auto) brigand::as_pack ( F &&  f)
constexpr

Call a functor with the types from a list.

Given a typelist List = tmpl::list<A, B, ...> (not necessarily with head tmpl::list), calls f as f(tmpl::type_<A>{}, tmpl::type_<B>{}, ...) and returns the result.

This is useful for converting a typelist into a parameter pack.

using List = tmpl::list<tmpl::size_t<1>, tmpl::size_t<2>, tmpl::size_t<3>>;
const size_t result = tmpl::as_pack<List>([](auto... args) {
return (... + tmpl::type_from<decltype(args)>::value);
});
CHECK(result == 6);
