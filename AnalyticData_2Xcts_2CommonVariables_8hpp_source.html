40 /*!
41  * \brief Implementations for variables that analytic-data classes can share
42  *
43  * Analytic-data classes can derive their variable computers from this class to
44  * inherit implementations for the `common_tags`. Note that some variables
45  * require a numeric differentiation. To compute those variables, a `mesh` and
46  * an `inv_jacobian` must be passed to the constructor. The `mesh` and the
47  * `inv_jacobian` can be set to `std::nullopt` if no variables with numeric
48  * derivatives are requested.
49  *
50  * \tparam DataType `double` or `DataVector`. Must be `DataVector` if variables
51  * with numeric derivatives are requested.
52  * \tparam Cache The `CachedTempBuffer` used by the analytic-data class.
53  */
54 template <typename DataType, typename Cache>
55 struct CommonVariables {
